        <section id="project" class="scroll-mt-24 fade-in-section">

            <div class="text-center mb-12"><div class="inline-block p-4 liquid-glass rounded-2xl mb-4 group transition-transform duration-300 hover:scale-110"><i data-lucide="scroll-text" class="w-10 h-10 gradient-icon"></i></div><h2 class="text-3xl md:text-4xl font-extrabold animate-title text-white" data-translate-key="projectTitle">El Proyecto de Grado</h2><p class="text-lg text-gray-400 mt-2 animate-subtitle" style="animation-delay: 0.2s;" data-translate-key="projectSubtitle">Desglosamos la fundación de la investigación: desde la identificación del problema y su justificación académica, hasta los objetivos específicos que guiaron el estudio y el marco teórico que lo sustenta.</p></div>

            <div class="grid md:grid-cols-2 lg:grid-cols-4 gap-8">

                <div class="liquid-glass p-6 rounded-2xl animated-card"><h3 class="font-bold text-xl mb-2 flex items-center text-slate-100"><i data-lucide="alert-circle" class="w-6 h-6 mr-2 gradient-icon"></i><span data-translate-key="projectCard1Title">El Problema</span></h3><p class="text-gray-300" data-translate-key="projectCard1Desc">Los estudiantes en prácticas profesionales enfrentan una doble carga de exigencias académicas y laborales, similar a la de un trabajador, lo que los convierte en una población vulnerable al síndrome de burnout y a sus consecuencias en la salud mental.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><h3 class="font-bold text-xl mb-2 flex items-center text-slate-100"><i data-lucide="check-check" class="w-6 h-6 mr-2 gradient-icon"></i><span data-translate-key="projectCard2Title">Justificación</span></h3><p class="text-gray-300" data-translate-key="projectCard2Desc">Ante la creciente deserción universitaria y los reportes de ansiedad y estrés, este estudio se vuelve crucial para proveer datos empíricos que permitan a la universidad crear estrategias efectivas de detección temprana, prevención e intervención.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><h3 class="font-bold text-xl mb-2 flex items-center text-slate-100"><i data-lucide="goal" class="w-6 h-6 mr-2 gradient-icon"></i><span data-translate-key="projectCard3Title">Objetivos</span></h3><p class="text-gray-300" data-translate-key="projectCard3Desc">El fin principal fue establecer la relación estadística entre el burnout académico (agotamiento, cinismo, ineficacia) y los problemas de salud mental (síntomas somáticos, ansiedad, disfunción social, depresión) en los practicantes.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><h3 class="font-bold text-xl mb-2 flex items-center text-slate-100"><i data-lucide="book-open" class="w-6 h-6 mr-2 gradient-icon"></i><span data-translate-key="projectCard4Title">Marco Teórico</span></h3><p class="text-gray-300" data-translate-key="projectCard4Desc">La investigación se ancla en los modelos teóricos de Maslach, Jackson y Schaufeli para definir el burnout académico, y en los constructos de Goldberg para evaluar la salud mental, adaptando estos marcos al contexto universitario colombiano.</p></div>

            </div>

        </section>

        

        <section id="method" class="scroll-mt-24 fade-in-section">

            <div class="text-center mb-12"><div class="inline-block p-4 liquid-glass rounded-2xl mb-4 group transition-transform duration-300 hover:scale-110"><i data-lucide="flask-conical" class="w-10 h-10 gradient-icon"></i></div><h2 class="text-3xl md:text-4xl font-extrabold animate-title text-white" data-translate-key="methodTitle">La Receta Científica</h2><p class="text-lg text-gray-400 mt-2 animate-subtitle" style="animation-delay: 0.2s;" data-translate-key="methodSubtitle">Detallamos el enfoque, diseño, población e instrumentos utilizados para garantizar que los resultados sean válidos, fiables y replicables.</p></div>

            <div class="grid md:grid-cols-2 lg:grid-cols-4 gap-8">

                <div class="liquid-glass p-6 rounded-2xl animated-card"><i data-lucide="search" class="w-8 h-8 gradient-icon mb-4"></i><h3 class="font-bold text-xl mb-2 text-slate-100" data-translate-key="methodCard1Title">Enfoque</h3><p class="text-gray-300" data-translate-key="methodCard1Desc">Se adoptó un enfoque <b>cuantitativo</b>, que se basa en la recolección y análisis de datos numéricos para contestar preguntas de investigación y probar hipótesis mediante la estadística.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><i data-lucide="file-spreadsheet" class="w-8 h-8 gradient-icon mb-4"></i><h3 class="font-bold text-xl mb-2 text-slate-100" data-translate-key="methodCard2Title">Diseño</h3><p class="text-gray-300" data-translate-key="methodCard2Desc"><b>No experimental, transversal y correlacional.</b> Se observaron los fenómenos tal como se dan en su contexto natural, en un único momento en el tiempo, para analizar su interrelación.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><i data-lucide="users" class="w-8 h-8 gradient-icon mb-4"></i><h3 class="font-bold text-xl mb-2 text-slate-100" data-translate-key="methodCard3Title">Población y Muestra</h3><p class="text-gray-300" data-translate-key="methodCard3Desc">La población fue de 104 estudiantes de Prácticas II de Psicología. Se calculó una muestra representativa de <b>82 participantes</b> para garantizar la validez externa de los hallazgos.</p></div>

                <div class="liquid-glass p-6 rounded-2xl animated-card"><i data-lucide="clipboard-check" class="w-8 h-8 gradient-icon mb-4"></i><h3 class="font-bold text-xl mb-2 text-slate-100" data-translate-key="methodCard4Title">Instrumentos</h3><p class="text-gray-300" data-translate-key="methodCard4Desc">Se utilizó el <b class="glossary-term" data-key="mbi">MBI-SS</b> (versión colombiana) para el Burnout y el <b class="glossary-term" data-key="ghq">GHQ-28</b> para Salud Mental, ambos test psicométricos validados que aseguran la fiabilidad de la medición.</p></div>

                const drawScatterPlot = () => {

                    svg.append("g")

                        .attr("class", "axis")

                        .call(d3.axisLeft(y));

                    

                    // Etiquetas de los ejes

                    svg.append("text")

                        .attr("class", "axis-label")

                        .attr("transform", "rotate(-90)")

                                            bar.style.transform = `translateX(calc(-100% + ${item.dataset.percent}%))`;

                                }

                                observer.unobserve(entry.target);

                            }

                        });

                    }, { threshold: 0.1 });

                    document.querySelectorAll('.fade-in-section').forEach(section => fadeInObserver.observe(section));


                    const navObserver = new IntersectionObserver((entries) => {

                        entries.forEach(entry => {

                            if (entry.isIntersecting && entry.intersectionRatio > 0.5) {

                                const id = entry.target.getAttribute('id');

                            }

                        });

                    }, { rootMargin: "-50% 0px -50% 0px", threshold: 0.5 });

                    document.querySelectorAll('main > section').forEach(section => navObserver.observe(section));
